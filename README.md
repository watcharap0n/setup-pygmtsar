## Installation

```bash
# Clone the repository
git clone https://github.com/watcharap0n/setup-pygmtsar.git
cd setup-pygmtsar

# Run the setup script for Linux
chmod +x install-pygmtsar.sh
./install-pygmtsar.sh
```
